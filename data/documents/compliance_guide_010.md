# Data Classification Standard / 数据分类标准

## Control Expectation / 控制要求


Handling requirements must match the classification level.
处理要求必须与数据分类等级相匹配。

## Levels / 等级


- Public: information approved for public release
- 公开：可对外发布的信息
- Internal: for employees and authorized contractors
- 内部：仅供员工和授权合作方使用
- Confidential: restricted to approved teams or roles
- 机密：限制在获批团队或角色范围内
- Restricted: highly sensitive information with named access control
- 受限：高度敏感信息，仅允许点名授权访问

## Purpose / 目的

This standard defines the classification levels used to manage company information securely.
本标准定义用于安全管理公司信息的数据分类等级。
