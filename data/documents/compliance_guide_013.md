# Encryption Key Handling / 加密密钥管理

## Key Requirements / 关键要求

- Keys must be stored in approved secure key management systems.
- 密钥必须存储在批准的安全密钥管理系统中。
- Access must be restricted to authorized personnel.
- 访问权限必须限制在获授权人员范围内。
- Key rotation must follow the approved schedule.
- 密钥轮换必须遵循已批准的计划。

## Logging / 日志要求

Key access and administrative actions must be logged.
密钥访问和管理操作必须留痕。

## Scope / 适用范围

This control governs the creation, storage, rotation, and revocation of encryption keys.
本控制适用于加密密钥的创建、存储、轮换和撤销。
